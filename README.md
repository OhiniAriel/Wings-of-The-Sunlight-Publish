# Wings-of-The-Sunlight-Publish
実は春休みから制作していました。
今はダンジョンをさまよってゴールを目指すだけです。
# 遊び方
**視点移動**: 左クリックでマウスが見えなくなります。それが合図だ。右クリックで視点移動モードが解除されるよ。
**移動**: WASD, 矢印キー  
**ダッシュ**: ctrlキー
**ジャンプ**: spaceキー, enterキー, Z  
**申し訳程度の画面ズーム**: shift
**決定**: enterキー, Z  
**キャンセル**: X, backspaceキー  
**メニュー**: E  
**なんかあったとき**: Qで強制終了  

他にもデバッグ機能は残しています。メイドインヘブンを起動するやつがあったりテストマップにいくやつだったり。
